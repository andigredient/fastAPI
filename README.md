# fastAPI

# Описание
Разработал сервис по сокращению ссылок с помощью FastAPI, postgresql и streamlit. Для большей наглядности использовал streamlit, где вся реализация поделена на 4 вкладки.
Реализовал следующие функции:

- Создание короткой ссылки
- Перенаправление на оригинальный URL
- Удаление ссылки
- Обновление URL
- Отображение статистики
- Создание кастомных ссылок
- Поиск ссылки по оригинальному URL
- Указание времени жизни ссылок с их удалением при истечении времени
- Создание БД без кэширования

  Дополнительно:
- Удаление неиспользуемых ссылок
- Применил сервис streamlit
